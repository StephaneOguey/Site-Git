Working Directory
Créer une branche Git Hub :
git init
Voir le status des fichiers d'une branche :
git status
Ajouter un fichier à la branche (Staged Area):
git add ..., git add --all
Versionner un fichier :
git commit -m "Nom de ce qu'on a fait"
Créer une autre branche :
git branch NomDeLaBranche
Changer de branche :
git checkout NomDeLaBranche
Fusion des branches (Ce mettre sur la branche qui reçois la fusion) :
git merge NomDeLaBranche à fusionner
Ce connecter à github

Ajouter
git remote add NomDeLaBranche AdresseDuSite
rapatrier les fichiers de local au serveur
git push -U NomDeLaBranche BrancheARapatrier

git pull NomDeLaBranche BrancheARapatrier
